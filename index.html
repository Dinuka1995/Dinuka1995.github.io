Char AND Code

This is a group project which is done as our 3rd year software Engineering project. As described by the name you can chat while coding using this software. Now you may wondering why does need to chat while coding? The idea is if you have an error while you coding and you couldn't find the bug then you have to get help from someone else. But its difficult without having an realtime interactive code editor. So that we provided a realtime interactive code editor and additionally video chating for better experience. We have used Node JS, MYSQL, html5 , boostrap , JQuery for this project.
